def quicksort(arr):
  if len(arr) <= 1: return arr
  pivot = arr[len(arr) // 2]
  left = [x for x in arr if x < pivot]
  middle = [x for x in arr if x == pivot]
  right = [x for x in arr if x > pivot]
  return quicksort(left) + middle + quicksort(right)
AI Explanation:
"This is a quicksort implementation using divide-and-conquer. It selects a pivot element, partitions the array into three parts (less than, equal to, and greater than pivot), then recursively sorts the left and right partitions..."
